# timestamp Renderer

Use `renderAs: 'timestamp'` for Unix-style timestamp values with controlled parse mode.

## Example

```tsx
{
  key: 'updatedAtEpoch',
  type: 'timestamp',
  renderAs: 'timestamp',
  renderAsOptions: {
    parseMode: 'epoch-sec',
    locale: 'en-US',
    timezone: 'utc',
    formatPreset: 'short',
  },
}
```

## Required Contract

- `renderAsOptions.parseMode` is required.
