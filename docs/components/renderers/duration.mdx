# duration Renderer

Use `renderAs: 'duration'` for elapsed-time values with unit-aware formatting.

## Example

```tsx
{
  key: 'latencySec',
  type: 'duration',
  renderAs: 'duration',
  renderAsOptions: {
    unit: 'sec',
    format: 'humanized',
    precision: 2,
    humanizedUnits: ['hour', 'minute', 'second'],
  },
}
```

## Required Contract

- `renderAsOptions.unit` is required.
