# time Renderer

Use `renderAs: 'time'` for time-only values with explicit parsing and timezone control.

## Example

```tsx
{
  key: 'checkpointTime',
  type: 'time',
  renderAs: 'time',
  renderAsOptions: {
    parseMode: 'iso-time',
    locale: 'en-US',
    timezone: 'utc',
    showSeconds: true,
    hour12: false,
  },
}
```

## Required Contract

- `renderAsOptions.parseMode` is required.
