# Built-in Icons

This page lists built-in icon names available in Widgemo.

Use these names anywhere config accepts an icon string (actions, zone icon config, and related icon-bearing surfaces).

For custom icon registration and override behavior, use [Custom Icons](/core/guides/custom-icons).

## Built-in Icon Names

### Actions

`plus`, `add`, `edit`, `delete`, `trash`, `view`, `copy`, `refresh`, `archive`, `share`, `export`, `download`, `upload`

### Navigation

`chevron-down`, `chevron-up`, `chevron-left`, `chevron-right`, `arrow-up`, `arrow-down`, `search`, `filter`, `sort`

### Views

`table`, `grid`, `board`, `carousel`, `calendar`

### UI

`settings`, `close`, `check`, `warning`, `message`, `clock`, `star`, `heart`, `users`, `user`, `chart`

### System

`ellipsis-vertical`, `external-link`, `book`

## Notes

- Built-in coverage can grow across releases.
- Prefer stable, explicit icon names in production configs.
- If you need branded or domain-specific icons, see [Custom Icons](/core/guides/custom-icons).
