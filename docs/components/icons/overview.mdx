# Icon Components Overview

Icons are used across actions, zones, badges, and other icon-capable UI surfaces.

## Icon Paths

- Built-in catalog and names: [Built-in Icons](/core/components/icons/built-in-icons)
- Registration and overrides: [Custom Icons](/core/guides/custom-icons)

## When to Use What

- Use built-in icon names for stable defaults.
- Use custom icons when you need branding, domain specificity, or alternate glyph sets.
