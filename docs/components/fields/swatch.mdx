# swatch Field Type

Use `type: 'swatch'` for color-like field values.

## Typical Use Cases

- Brand color tokens
- Category color coding
- Theme preview values

## Example

```tsx
{ key: 'brandColor', label: 'Brand Color', type: 'swatch' }
```
