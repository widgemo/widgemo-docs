# number Field Type

Use `type: 'number'` for raw numeric values when you do not need specialized renderer semantics.

## Typical Use Cases

- Counts and totals
- Baseline metric columns
- Numeric sorting fields

## Example

```tsx
{ key: 'count', label: 'Count', type: 'number' }
```
