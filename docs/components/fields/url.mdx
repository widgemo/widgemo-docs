# url Field Type

Use `type: 'url'` for web addresses and endpoint links.

## Typical Use Cases

- Profile URLs
- Document links
- External dashboard links

## Example

```tsx
{ key: 'profileUrl', label: 'Profile', type: 'url' }
```
