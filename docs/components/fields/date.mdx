# date Field Type

Use `type: 'date'` for calendar-date values.

## Typical Use Cases

- Settlement dates
- Due dates
- Posting dates

## Example

```tsx
{
  key: 'settlementDate',
  label: 'Settlement Date',
  type: 'date',
  renderAs: 'date',
  renderAsOptions: { parseMode: 'iso-date', formatPreset: 'short' },
}
```

## Contract

- Pair with `renderAs: 'date'` for explicit temporal rendering.
- `renderAsOptions.parseMode` is required in temporal renderer mode.
