# datetime Field Type

Use `type: 'datetime'` for combined date and time values.

## Typical Use Cases

- Event timestamps
- Posting times
- Workflow activity times

## Example

```tsx
{
  key: 'postedAt',
  label: 'Posted At',
  type: 'datetime',
  renderAs: 'datetime',
  renderAsOptions: { parseMode: 'iso-datetime', formatPreset: 'short' },
}
```

## Contract

- Pair with `renderAs: 'datetime'` for explicit temporal rendering.
- `renderAsOptions.parseMode` is required in temporal renderer mode.
