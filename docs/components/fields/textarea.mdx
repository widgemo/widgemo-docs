# textarea Field Type

Use `type: 'textarea'` for longer text content where multi-line values are expected.

## Typical Use Cases

- Notes and comments
- Summaries and descriptions
- Internal audit annotations

## Example

```tsx
{ key: 'notes', label: 'Notes', type: 'textarea' }
```
