# Field Components Overview

Field components are the field-level behavior and semantics that remain stable across modes.

## Choose the Right Layer

- Use `type` when the field is conceptually a known data category.
- Use `renderAs` when you need specific visual presentation.
- Use custom field types when domain behavior should be reusable and first-class.

If both are present, `renderAs` controls display while `type` still communicates semantic intent.
