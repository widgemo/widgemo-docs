# relation Field Type

Use `type: 'relation'` for values representing linked entities or references.

## Typical Use Cases

- Owner or assignee fields
- Parent-child references
- Entity linkage metadata

## Example

```tsx
{ key: 'owner', label: 'Owner', type: 'relation' }
```
