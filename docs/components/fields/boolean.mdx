# boolean Field Type

Use `type: 'boolean'` for true/false values with optional fallback labels.

## Typical Use Cases

- Feature flags
- Active/inactive state
- Binary checks

## Example

```tsx
{
  key: 'isActive',
  label: 'Active',
  type: 'boolean',
  booleanTrueLabel: 'Yes',
  booleanFalseLabel: 'No',
}
```

## Notes

- `booleanTrueLabel` and `booleanFalseLabel` apply to boolean fallback rendering.
- For richer visual semantics, combine with a renderer.
