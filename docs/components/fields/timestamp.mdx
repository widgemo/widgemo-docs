# timestamp Field Type

Use `type: 'timestamp'` for epoch-style timestamp values.

## Typical Use Cases

- Event ingestion times
- Audit entries
- Source-system Unix timestamp fields

## Example

```tsx
{
  key: 'updatedAtEpoch',
  label: 'Updated',
  type: 'timestamp',
  renderAs: 'timestamp',
  renderAsOptions: { parseMode: 'epoch-sec', formatPreset: 'short' },
}
```

## Contract

- Pair with `renderAs: 'timestamp'` for explicit temporal rendering.
- `renderAsOptions.parseMode` is required in temporal renderer mode.
