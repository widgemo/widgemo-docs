# Mode Components Overview

Modes control the full content surface for a widget, including layout, interaction model, and information density.

## Built-in Modes

- [Table Mode](/core/concepts/modes/table)
- [Grid Mode](/core/concepts/modes/grid)
- [Carousel Mode](/core/concepts/modes/carousel)
- [Board Mode](/core/concepts/modes/board)
- [Chart Mode](/core/concepts/modes/chart)
- [Responsive Switching](/core/concepts/modes/responsive-switching)

## How to Choose

- Choose a mode first for the overall surface behavior.
- Then choose field types and renderers for per-field semantics and visual treatment.
