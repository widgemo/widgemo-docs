## Getting Started at a Glance

This intro is a fast orientation. The steps below are the absolute minimum to render Widgemo. For setup specifics and edge cases, use the Getting Started pages.

1. Install widgemo-core.
2. Import the stylesheet once at your app root.
3. Render `Widgemo` with `data` and a `zones`-based `config`.

```bash
npm install @widgemo/widgemo-core
```

```tsx
import { WidgemoThemeProvider, Widgemo } from '@widgemo/widgemo-core';
import '@widgemo/widgemo-core/style.css';

function App() {
  return (
    <WidgemoThemeProvider theme="light">
      <Widgemo data={data} config={config} />
    </WidgemoThemeProvider>
  );
}
```
