# widgemo-core Documentation

Welcome to the **widgemo-core** documentation hub. Here you'll find everything you need to build, customize, and extend zero-dependency React components for configurable data display.

## What is widgemo-core?

**widgemo-core** is a zero-dependency React component library that makes it easy to build powerful, configurable data components. With minimal runtime dependencies (React and React-DOM are peers), it provides:

- **Multiple rendering modes** — Table, Grid, Carousel, Board, and Chart modes
- **Flexible configuration** — Define component behavior entirely through a JSON-like config object
- **Built-in field renderers** — Currency, progress, links, ratings, and more, out of the box
- **Automatic theming** — Light/dark mode support with customizable CSS variables
- **Extension API** — Register custom renderers, modes, icons, and field types
- **TypeScript-first** — Complete type safety from the ground up

## Getting Started (2 minutes)

Install via npm:

```bash
npm install @widgemo/widgemo-core react react-dom
```

Render a table in your React app:

```tsx
import { Widgemo, WidgemoThemeProvider } from '@widgemo/widgemo-core';

export function MyDataTable() {
  const data = [
    { id: 1, name: 'Alice', role: 'Engineer', salary: 125000 },
    { id: 2, name: 'Bob', role: 'Designer', salary: 95000 },
  ];

  return (
    <WidgemoThemeProvider>
      <Widgemo
        config={{
          mode: 'table',
          fields: [
            { name: 'name', label: 'Name' },
            { name: 'role', label: 'Role' },
            { name: 'salary', label: 'Salary', renderAs: 'currency' },
          ],
          content: {
            filtering: [{ fieldKey: 'role', operator: 'eq', value: 'Engineer' }],
            searching: { enabled: true },
            sorting: { enabled: true },
          },
        }}
        data={data}
      />
    </WidgemoThemeProvider>
  );
}
```

That's it! You now have a fully-featured data table with search, filter, and sort.

## Peer Dependencies

widgemo-core requires **React 16.8+** and **React-DOM 16.8+**. No other runtime dependencies.

```json
{
  "peerDependencies": {
    "react": "^16.8.0",
    "react-dom": "^16.8.0"
  }
}
```
