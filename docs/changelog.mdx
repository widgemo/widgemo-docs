## [0.1.0] - 2026-04-22

Initial public release.

### Core Features

- `<Widgemo>` component with full TypeScript support
- Five display modes: `table`, `grid`, `carousel`, `board` (with swimlanes), and responsive switching
- Zone-based layout: `zones.header`, `zones.content`, `zones.footer`
- `WidgemoConfig` with `interactions.onEvent` global event bus
- Per-item and zone-level `ActionConfig` with `placement: 'pinned' | 'onHover' | 'menu'`
- `ActionOverflowConfig` with responsive `maxInline` per breakpoint
- Board mode drag-and-drop with swimlane matrix support
- `ContentConfig` search, sorting, filtering, grouping, and pagination
- Nine built-in `renderAs` renderers: `badge`, `currency`, `progress`, `rating`, `link`, `jsonPreview`, `deltaValue`, `sparkTrend`, `scoreMeter`
- `WidgemoThemeProvider` with `'light' | 'dark' | 'auto' | WidgemoTheme`
- CSS variable overrides via `--widgemo-*`
- `widgemoRegistry` extension APIs for renderers, modes, hooks, icons, and field types
- `devMode` inspector overlay with configurable docs link
- `InteractionContext` with board drag context (`from` and `to`)
- Card layout types: `auto`, `flex`, `grid` template areas, `sections`
- Loading and error states with skeleton indicator and retry support
- Responsive mode switching with breakpoint-scoped config overrides
