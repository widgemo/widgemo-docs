# Theme API

Theme-related exports from `@widgemo/widgemo-core`:

## `WidgemoThemeProvider`

Wrap your component tree to apply a theme:

```tsx
import { WidgemoThemeProvider } from '@widgemo/widgemo-core';

// Preset
<WidgemoThemeProvider theme="dark">…</WidgemoThemeProvider>

// System preference
<WidgemoThemeProvider theme="auto">…</WidgemoThemeProvider>

// Object
<WidgemoThemeProvider theme={{ colors: { actionButtonBg: '#0f172a' } }}>…</WidgemoThemeProvider>
```

`theme="auto"` is evaluated at render time from `window.matchMedia('(prefers-color-scheme: dark)')` and does not subscribe to live OS preference changes.

Full widget dark/light theming is guaranteed through `WidgemoThemeProvider`.

## `useWidgemoTheme`

Access the resolved theme inside any component:

```tsx
import { useWidgemoTheme } from '@widgemo/widgemo-core';

function MyHeader() {
  const theme = useWidgemoTheme();
  return <header style={{ background: theme.zone?.backgroundColor }} />;
}
```

Without a provider, `useWidgemoTheme()` returns an empty theme object (`{}`).

## `registerTheme` / `getTheme`

Register named themes for reuse:

```tsx
import { registerTheme, getTheme } from '@widgemo/widgemo-core';

registerTheme('my-dark-theme', {
  colors: {
    actionButtonBg: '#0f172a',
    actionButtonColor: '#f8fafc',
  },
});

const theme = getTheme('my-dark-theme');

// then reference it in config:
const config = {
  theme: 'my-dark-theme',
  zones: { /* … */ },
};
```

`config.theme` contributes to runtime zone/action theme layering. Use `WidgemoThemeProvider` for full-surface dark/light theming.

## `mergeZoneTheme`

Merge partial zone theme overrides safely:

```tsx
import { mergeZoneTheme } from '@widgemo/widgemo-core';

const merged = mergeZoneTheme(baseTheme, overrides);
```

`mergeZoneTheme` performs a safe shallow merge for zone theme objects: override keys win, unchanged base keys are preserved, and an undefined base is handled safely.

## `defaultWidgemoTheme`

Provider light preset alias (`defaultLightTheme`) exported from the public API. Useful as a base for custom theme objects:

```tsx
import { defaultWidgemoTheme } from '@widgemo/widgemo-core';

console.log(defaultWidgemoTheme);
```

## `WidgemoTheme` Interface

```ts
interface WidgemoTheme {
  colors?: {
    actionButtonBg?: string;
    actionButtonColor?: string;
    actionButtonBorder?: string;
    actionButtonHoverBg?: string;
    actionButtonHoverBorder?: string;
    actionMenuBg?: string;
    actionMenuColor?: string;
    actionMenuItemHoverBg?: string;
    actionMenuItemHoverColor?: string;
    [key: string]: string | undefined;
  };
  spacing?: Record<string, string>;
  zone?: ZoneTheme;
  action?: ActionTheme;
}
```
