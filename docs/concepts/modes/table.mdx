# Table Mode

modeConfig.table controls table-specific presentation and table lifecycle hooks.

Shared content features such as search, pagination, filtering, sorting descriptors, actions, gestures, and responsive breakpoints are documented in [Content Config](/core/reference/content-config).

## Table Option Reference

| Option | Possible values | Default | Effect |
|---|---|---|---|
| type | traditional or rich-cells | traditional | Chooses classic row/cell table layout or richer multi-column cell blocks. |
| columns | any number >= 1 | 1 | Number of rich-cell columns when type is rich-cells. |
| hover | true or false | true | Enables row hover behavior. |
| showHeader | true or false | true | Shows or hides the header row. |
| alternatingRows | true or false | true | Applies zebra striping to alternating rows. |
| rowSeparator | true or false | true | Draws row bottom borders. |
| conditionalBackgroundColor | function returning a color string, or an object with backgroundColor and optional color | unset | Applies dynamic row background and optional text color per entity. |
| hooks.onSort | function (field, direction) => void | unset | Fires when a column sort changes. |
| hooks.onGroup | function (field) => void | unset | Fires when grouping changes. |
| hooks.preRowRender | function (entity) => entity | unset | Lets you transform a row before render. |
| hooks.onGroupToggle | function (groupValue, isExpanded) => void | unset | Fires when a group expands or collapses. |

## Grouping UI Controls

Grouping is configured under `zones.content.groupings`; the canonical contract and cross-mode behavior live in [Content Config: Grouping](/core/reference/content-config#grouping).

This section is intentionally table-specific and focuses on grouping interaction surfaces that apply in table mode.

| Option | Possible values | Default | Effect |
|---|---|---|---|
| fieldKey | any field key | required | Field used to build groups. |
| initiallyCollapsed | true or false | false | Sets whether groups start collapsed. |
| collapsible | true or false | true | Lets users expand or collapse group blocks. |
| showDropdownControl | true or false | false | Shows the group-by dropdown control above the table. |
| showHeaderControls | true or false | true | Shows grouping icons in header cells for groupable fields. |

Dropdown visibility still requires at least one field with groupable: true so the control has selectable columns.

## Coverage Summary

Every table setting is represented by at least one live example on this page:

- type and columns: Complete Table Mode Configuration, Rich Cells
- hover, showHeader, alternatingRows, rowSeparator: Traditional Table, Presentation Toggles
- conditionalBackgroundColor: Row Highlighting Example, Complete Table Mode Configuration
- groupings.showDropdownControl and groupings.showHeaderControls: Grouping Control Variants
- hooks.onSort, hooks.onGroup, hooks.preRowRender, hooks.onGroupToggle: Hook Callback Example, Complete Table Mode Configuration

For field-level table behavior such as sortable, width, and align, see [Field Config](/core/reference/field-config).
