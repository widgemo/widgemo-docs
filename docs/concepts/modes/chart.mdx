# Chart Mode

## Mode Purpose and When to Use

Chart mode owns the full content area as a chart.
Use it when the visualization itself is the primary surface, not a small field renderer.
For inline composition bars, see [compositionBar vs Chart Mode](/core/guides/composition-bar#when-to-use-compositionbar-vs-chart-mode).

`modeConfig.chart` accepts five series types: `bar`, `line`, `area`, `pie`, and `donut`. The chart can also control legend placement, grid lines, labels, size, colors, tooltip behavior, and line/area/polar tuning.

## Mode Option Reference

| Option | Values | Default | Effect |
|---|---|---|---|
| `xAxis` | string | unset | Selects the label field used for chart categories. |
| `height` | number | `300` | Sets the chart height. |
| `showLegend` | `true`, `false` | unset | Shows or hides the legend. |
| `legendAlign` | `left`, `center`, `right` | `center` | Aligns the legend left, center, or right. |
| `showGrid` | `true`, `false` | unset | Shows or hides the cartesian grid. |
| `showLabels` | `true`, `false` | unset | Shows or hides data labels. |
| `colors` | string array | unset | Overrides the default series color palette. |
| `orientation` | `vertical`, `horizontal` | `vertical` | Flips bar charts to horizontal when set to `horizontal`. |
| `lineThickness` | number | `3.5` | Sets the default thickness for line and area series. |
| `showDots` | `true`, `false` | `false` | Shows or hides line/area data points. |
| `areaGradient` | `true`, `false` | `true` | Enables or disables gradient fills for area series. |
| `donutInnerRadiusRatio` | number (`0.2` to `0.9`) | `0.58` | Controls the donut hole size. |
| `tooltip.position` | `top`, `bottom`, `left`, `right`, `top-right`, `top-left`, `bottom-right`, `bottom-left` | `top-right` | Sets tooltip placement. |
| `tooltip.render` | `(context) => ReactNode` | unset | Replaces the built-in tooltip content. |

### Example: Horizontal Bars

If you want a horizontal bar chart, keep the series type as `bar` and set `orientation: 'horizontal'`.

```tsx
chart: {
  xAxis: 'month',
  orientation: 'horizontal',
  series: [{ key: 'revenue', type: 'bar' }],
}
```

## Mode-Specific Caveats

- `orientation` applies only to bar series.
- `donutInnerRadiusRatio` applies only to donut series.
