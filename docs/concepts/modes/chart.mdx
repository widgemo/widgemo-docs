# Chart Mode

Chart mode owns the full content area as a chart. Use it when the visualization itself is the primary surface, not a small field renderer. For inline composition bars, see [compositionBar vs Chart Mode](/core/guides/composition-bar#when-to-use-compositionbar-vs-chart-mode).

`modeConfig.chart` accepts five series types: `bar`, `line`, `area`, `pie`, and `donut`. The chart can also control legend placement, grid lines, labels, size, colors, tooltip behavior, and line/area/polar tuning.

Shared content behavior such as search, actions, filtering, pagination, and responsive breakpoints is documented in [Content Config](/core/reference/content-config).

## Bar Chart

Use bar charts for direct comparisons across categories.

  title="Bar Chart"

  config={barChartConfig}
/>

<div style={{ marginBottom: '1rem' }} />

```tsx
const config = {
  zones: {
    content: {
      mode: 'chart',
      modeConfig: {
        chart: {
          xAxis: 'month',
          height: 320,
          showLegend: true,
          showGrid: true,
          series: [
            { key: 'revenue', label: 'Revenue', type: 'bar' },
            { key: 'visits', label: 'Visits', type: 'bar' },
          ],
        },
      },
    },
  },
};
```

## Chart Options

| Option | What it changes |
|---|---|
| `xAxis` | Selects the label field used for chart categories. |
| `height` | Sets the chart height. |
| `showLegend` | Shows or hides the legend. |
| `legendAlign` | Aligns the legend left, center, or right. |
| `showGrid` | Shows or hides the cartesian grid. |
| `showLabels` | Shows or hides data labels. |
| `colors` | Overrides the default series color palette. |
| `orientation` | Flips bar charts to horizontal when set to `horizontal`. |
| `lineThickness` | Sets the default thickness for line and area series. |
| `showDots` | Shows or hides line/area data points. |
| `areaGradient` | Enables or disables gradient fills for area series. |
| `donutInnerRadiusRatio` | Controls the donut hole size. |
| `tooltip.position` | Sets tooltip placement. |
| `tooltip.render` | Replaces the built-in tooltip content. |

### Example: Horizontal Bars

If you want a horizontal bar chart, keep the series type as `bar` and set `orientation: 'horizontal'`.

```tsx
chart: {
  xAxis: 'month',
  orientation: 'horizontal',
  series: [{ key: 'revenue', type: 'bar' }],
}
```
